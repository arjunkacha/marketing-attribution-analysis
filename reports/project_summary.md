# Project Summary

## Objective

To evaluate marketing channel performance using different attribution models and visualize business KPIs through an interactive Power BI dashboard.

## Data Sources

- customer_journey.csv
- conversions.csv
- marketing_spend.csv

## Analysis Performed

1. First Touch Attribution
2. Last Touch Attribution
3. Linear Attribution
4. ROAS Calculation
5. CAC Calculation

## Dashboard Metrics

- Total Revenue
- Total Marketing Spend
- Total Conversions
- Average ROAS
- Average CAC

## Conclusion

The analysis highlights how attribution methodology impacts revenue allocation across marketing channels and helps identify the most efficient acquisition sources.
